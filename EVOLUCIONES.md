# Evoluciones

Las evoluciones permiten mejorar una carta del club.

## Funcionamiento

El usuario elige una carta y se aplica una mejora sencilla:

- +2 media
- +2 ritmo
- +2 tiro
- +2 pase
- +1 regate
- +1 defensa
- +1 fisico

Nunca se pasa de 99.

## Endpoints

```text
GET /api/evoluciones
POST /api/evoluciones/carta/{cartaId}
```
