# Criterios de Aceptación
1. El código no debe romper nada de lo ya existente (si funciona).
2. Deben estar completos todos los endpoints CRUD y los de filtros.
3. El frontend debe mostrar la información correcta usando el backend.
4. Los tests unitarios deben pasar con éxito.
5. Todo el proyecto debe poder levantarse siguiendo las instrucciones del README.
